# Student Marks Analysis & Grade Predictor

## Project Overview

This project focuses on analyzing student academic performance and predicting math scores using Machine Learning techniques.

The project includes:

* Data Cleaning
* Feature Engineering
* Data Visualization
* Student Performance Analysis
* Linear Regression Model
* Actual vs Predicted Score Comparison

The main goal is to understand factors affecting student performance and build a simple predictive model.

---

## Dataset Used

### Dataset Name

Students Performance in Exams Dataset

### Source

https://www.kaggle.com/datasets/spscientist/students-performance-in-exams

### Dataset Features

The dataset contains:

* Gender
* Race/Ethnicity
* Parental Level of Education
* Lunch Type
* Test Preparation Course
* Math Score
* Reading Score
* Writing Score

## Machine Learning Model

### Model Used

Linear Regression

### Target Variable

Math Score

### Input Features

* Reading Score
* Writing Score

### Evaluation Metrics

* Mean Absolute Error (MAE)
* R2 Score

The model predicts student math performance based on reading and writing scores.

---

## Results

The project successfully identifies important performance patterns among students.

Students who completed test preparation and had stronger reading/writing performance generally achieved better math scores.

The prediction model produced good accuracy and meaningful results.
